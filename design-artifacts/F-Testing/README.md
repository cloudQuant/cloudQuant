# Testing Strategy

This directory contains testing strategy documents generated by the TEA workflow.

## What is a Testing Strategy?

A Testing Strategy includes:
- Test coverage requirements
- Test types (unit. integration. e2e)
- Testing tools and frameworks
- Quality gates

## Template

Testing strategies follow the template defined in `_bmad/tea/workflows/testarch/`.
