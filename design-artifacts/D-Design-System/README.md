# Design System

This directory contains design system documentation generated by the WDS workflow.

## What is a Design System?

A Design System includes:
- Component library specifications
- Design tokens (colors. spacing. typography)
- UI patterns and guidelines
- Accessibility standards

## Template

Design system docs follow the template defined in `_bmad/wds/workflows/7-design-system/`.
