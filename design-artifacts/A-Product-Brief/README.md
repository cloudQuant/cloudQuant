# Product Brief

This directory contains product brief documents generated by the WDS workflow.

## What is a Product Brief?

A Product Brief is a high-level document that defines:
- Product vision and goals
- Target users and market
- Core features and requirements
- Success criteria

## Template

Product briefs follow the template defined in `_bmad/wds/workflows/0-project-setup/`.
