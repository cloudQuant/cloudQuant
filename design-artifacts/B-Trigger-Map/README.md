# Trigger Map

This directory contains trigger map documents generated by the WDS workflow.

## What is a Trigger Map?

A Trigger Map defines:
- User psychology triggers
- Business goal mappings
- User motivation patterns
- Key decision points

## Template

Trigger maps follow the template defined in `_bmad/wds/workflows/0-trigger-mapping/`.
