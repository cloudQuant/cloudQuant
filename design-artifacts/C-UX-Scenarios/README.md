# UX Scenarios

This directory contains UX scenario documents generated by the WDS workflow.

## What are UX Scenarios?

UX Scenarios define:
- User interaction flows
- Step-by-step user journeys
- UI state transitions
- Edge cases and error handling

## Template

UX scenarios follow the template defined in `_bmad/wds/workflows/0-scenarios/`.
