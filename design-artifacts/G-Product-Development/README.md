# Product Development

This directory contains development plan documents generated by the BMM workflow.

## What is Product Development Planning?

Product Development planning includes:

- Sprint plans
- Epic breakdowns
- Task assignments
- Timeline estimates

## Template

Development plans follow the template defined in `_bmad/bmm/workflows/sprint-planning/`.
