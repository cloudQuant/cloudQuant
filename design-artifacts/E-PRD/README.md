# Product Requirements Document (PRD)

This directory contains PRD documents generated by the BMM workflow.

## What is a PRD?

A Product Requirements Document defines:

- Feature requirements
- User stories
- Acceptance criteria
- Technical constraints

## Template

PRDs follow the template defined in `_bmad/bmm/workflows/1-prd/`.
