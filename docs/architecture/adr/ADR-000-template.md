# ADR Template

Use this template to create new Architecture Decision Records.

## Template

```markdown
# ADR-XXX: [Short Title]

| Status | Date | Decision Makers |
|--------|------|-----------------|
| [Proposed/Accepted/Deprecated] | YYYY-MM-DD | [Names] |

## Context
[Describe the context and problem statement]

## Decision
[Describe the decision that was made]

## Consequences
### Positive
- [List positive impacts]

### Negative
- [List potential negative impacts or risks]

### Alternatives Considered
- [List alternatives that were considered and why they were not chosen]

## References
- [Related links or references]
```

## Creating a New ADR

1. Copy this template
2. Rename to `ADR-XXX-short-title.md` (increment XXX)
3. Fill in the sections
4. Submit for review
