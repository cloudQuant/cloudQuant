# 架构决策记录 (ADR) 索引

本目录记录 cloudQuant 项目的重要架构决策。

## 什么是 ADR?

架构决策记录 (Architecture Decision Record) 是一种记录重要架构决策的文档模式。每个 ADR 描述：

- **背景**: 决策的背景和上下文
- **决策**: 做出的具体决定
- **后果**: 决策带来的影响和结果

## ADR 模板

```markdown
# ADR-NNN: [决策标题]

## 状态

[提议中 | 已采纳 | 已废弃 | 已替代]

## 上下文

[描述决策的背景和需要解决的问题]

## 决策

[描述做出的决定及其理由]

## 后果

### 正面影响

- [列出正面影响]

### 负面影响

- [列出可能的负面影响或风险]

### 替代方案

- [列出考虑过的其他方案及为何没有选择]

## 参考

- [相关链接或参考资料]
```
