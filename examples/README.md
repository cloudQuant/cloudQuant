# cloudQuant Examples

This directory contains example scripts demonstrating how to use the cloudQuant framework.

## Examples

### 1. Simple Backtest (`simple_backtest.py`)

A basic example showing how to:
- Create a trading strategy
- Initialize a backtest engine
- Run a backtest
- Display results

```bash
python examples/simple_backtest.py
```
